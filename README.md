# nicehash_domain_whitelist

# What is this?
A whitelist of domains for users that block cryptominer domains using a blacklist but want to use nicehash.

# How do I add these domains to pi-hole?
Open up the pi-hole UI, navigate to whitelist (http://pi.hole/admin/list.php?l=white), then paste the following text into the 'Add a domain' field and click add. (Updated 2018-03-16)
